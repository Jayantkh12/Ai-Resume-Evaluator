# Ai-Resume-Evaluator
AI Resume Evaluator is a Flask-based web application that uses Google Gemini AI to analyze resumes against job descriptions and provide an ATS-style evaluation. Users can upload their resume and paste a job description to receive an instant match score, missing keywords, strengths, weaknesses, and a final verdict.

This project is designed to help students and job seekers improve their resumes and increase their chances of passing Applicant Tracking Systems (ATS).

Features
    ATS Match Score Calculation
    Missing Keywords Detection
    Resume Strength & Weakness Analysis
    PDF Resume Upload Support
    Responsive and Modern UI
    Google Gemini AI Integration
Technologies Used
    Python
    Flask
    HTML/CSS/JavaScript
    Google Gemini API
    PyPDF2
    dotenv
